题目：删掉一个元素以后全为 1 的最长子数组 给你一个二进制数组 nums ，你需要从中删掉一个元素。请你在删掉元素的结果数组中，返回最长的且只包含 1 的非空子数组的长度。如果不存在这样的子数组，请返回 0。 该题关键是想明白滑动窗口在什么情况下滑动，也就是当判断带该区间有2个0时，那么滑动窗口就开始滑动使得区间只含1个0。在代码里sum可以看作是有几个1，而right-left-1表示区间里含有2个0。

func longestSubarray(nums []int) int {
left, sum, res := 0,0,0
for right := 0 ; right < len(nums) ; right++ {
sum += (nums[right] & 1) //遇到1相加
** //right-left-1表示该区间个数减去2个0之后剩余的1的个数**
for left < right && sum <= right - left - 1{
if nums[left] == 1 {
sum–
}
left++
}
res = max(res, right-left)
}
return res
}

return a
